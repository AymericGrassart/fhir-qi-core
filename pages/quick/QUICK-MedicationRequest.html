MedicationRequest

Profile of medication prescription, indicating which elements must be supported in rule and measure interpreters.

= Must Support, = Is Modifier, = QiCore defined extension

Binding: A code specifying the state of the prescribing event. Describes the lifecycle of the prescription. (required)

Binding: The kind of medication order (required)

Binding: A coded concept identifying where the medication ordered is expected to be consumed or administered (preferred)
